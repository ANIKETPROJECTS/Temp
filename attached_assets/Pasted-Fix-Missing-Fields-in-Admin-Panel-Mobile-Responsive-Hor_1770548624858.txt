Solution: Horizontal Scroll Table for Mobile
Implement a horizontally scrollable table on mobile devices that:

Implementation Approach:
css/* Mobile responsive styles for queue tables */
@media (max-width: 768px) {
  .queue-table-container {
    overflow-x: auto;
    -webkit-overflow-scrolling: touch; /* Smooth scrolling on iOS */
    width: 100%;
  }
  
  .queue-table {
    width: 100%;
    min-width: 900px; /* Ensures all columns fit, forces horizontal scroll */
    display: table;
  }
  
  /* Optional: Add scroll hint/shadow */
  .queue-table-container::after {
    content: '';
    position: absolute;
    right: 0;
    top: 0;
    bottom: 0;
    width: 20px;
    background: linear-gradient(to right, transparent, rgba(0,0,0,0.1));
    pointer-events: none;
  }
  
  /* Ensure columns don't shrink too much */
  .queue-table th,
  .queue-table td {
    white-space: nowrap;
    min-width: 80px; /* Adjust as needed per column */
  }
  
  /* Specific column widths for readability */
